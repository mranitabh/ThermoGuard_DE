# ThermoGuard_DE

ThermoGuard_DE is a machine learning project focused on predictive thermal monitoring and risk detection.

## Project Overview

This notebook performs data loading, preprocessing, exploratory data analysis, model development, and evaluation for thermal risk prediction.

## Dataset

The dataset is loaded directly inside the notebook from an online source.  
No local `data.csv` file is required.

## How to Run

1. Open the notebook `ThermoGuard_DE_Lipy...ipynb`
2. Click **Open in Colab** or download and upload it to Google Colab
3. Run all cells
4. The dataset will load automatically from the online source used inside the notebook

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab
- Jupyter Notebook
